# Limpeza de dados para campanhas de marketing de Instituições Financeiras
Esse um exercício realizado no capítulo de Limpeza de Dados do curso Análise de Dados com Linguagem Python promovido Data Science Academy (DSA). A atividade se trata da limpeza de dados de marketing de Instituições Financeiras. Ao fim, alguns gráficos foram criados com as informações disponíveis.

Site da DSA: https://www.datascienceacademy.com.br/
